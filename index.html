MOVIECART

ENJOY UNLIMITED VIDEOS FOR FREE

Moviecart is a video streaming app which uses Server-client model. It uses VLCJ API for streaming video. It provides other types of services like- login, signup, upload, showing video list. A client can use our app in 2 modes- User mode and Guest mode. In guest mode a client don't have permission to upload any video.
